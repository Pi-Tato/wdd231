Game Development Guide

Choose Your Game Engine

There are a lot of game engines to choose from. Here are some of the most popular ones:

  • Unity
  • Unreal Engine
  • Godot
  • Construct 3

Each engine has its own advantages and disadvantages, so it's important to choose the one that best suits your needs. For this guide, we'll be using Unity.

Getting Started with Unity

Before you can start coding, you need to install Unity. You can download it from the Unity's website. unity.com/download

Creating a New Project

Once you have Unity installed, you can create a new project. Open the Unity Hub and click the "New Project" button on top right.

Under "Project Name", enter a name for your game. Then under "Project Location", enter a location for your project. The "Unity Organization" will be your username.

There are a lot of Render Pipelines, but we will be using the Universal 3D Render Pipeline (URP). Then click "Create project" on the bottom right.

Unity will then create a new project in the location you specified. It will take a few minutes to open your new project initially.

When your project opens up, you may see errors in the console. This is normal and can be ignored. To fix it, delete "Sample Scene" in the "Scenes" folder. Then make a new scene in the "Scenes" folder, close Unity and open it again. The errors should be gone.

Creating a New Scene

After you have created a new scene, let's start on Creating a Main Menu!

Right click on the Hierarchy and click "UI" then click "Panel". Then name it "Main Menu". This will create a "Canvas" with a "Panel" inside of it.
